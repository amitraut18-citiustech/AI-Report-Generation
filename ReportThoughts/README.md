# ReportThoughts (Phase 1a output)

Reviewable business-logic analysis of each legacy report, one `*.thought.md` per report,
produced by the `report-researcher` agent (`/report-research`).

**Review these before migrating.** A developer confirms Claude's understanding here —
correcting a misread formula in a thought file is far cheaper than debugging generated
HTML. Once approved, run `/report-migrate` to generate the static reports.
