# HTMLReportsFolder (Phase 1b output)

Static report replacements generated by the `report-migrator` agent (`/report-migrate`)
from approved thought files. Three files per report:

- `{report_key}.html` — self-contained template with data-binding hooks
- `{report_key}.js` — renders from `window.REPORT_DATA` (no network calls)
- `{report_key}.md` — human-facing documentation

At runtime the .NET app injects `window.REPORT_DATA` (parameters, rows, narrative, meta)
and serves the populated HTML. See
[`.claude/skills/report-forge/references/html-report-template.md`](../.claude/skills/report-forge/references/html-report-template.md).
